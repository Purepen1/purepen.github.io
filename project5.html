AUTOMATE CRYPTO API PULL USING PYTHON

Python code is making an API request to the CoinMarketCap API to fetch the latest cryptocurrency market data. The code is using the `requests` library to make the API request and fetch the response, and the 'json' library to parse the response into JSON format. Then, it is using the `pandas` library to convert the JSON response into a dataframe and do some analysis on the data. The code starts by setting up the API request URL, headers, and parameters. It then creates a 'Session object with the headers, sends a GET request to the API URL with the parameters, and loads the JSON response into a dictionary with `json.loads()'.

pThe `pd.json_normalize() method is used to convert the dictionary to a dataframe, and then `pd.set_option()`` is used to set options for displaying all columns and rows of the dataframe. The code defines a function `api_runner()` that does the same thing as the initial code block, but also adds the new data to the existing dataframe. The code then enters a `for loop that runs the 'api_runner()` function every 60 seconds for 333 iterations. The data is then written to a CSV file and read back in as a new dataframe `df72`.

The code goes on to calculate the mean of the percent change columns for each coin, stack them into a single column, and convert the resulting series into a dataframe. It then sets a new index for this dataframe and renames one of the columns.
